## Installation

Add `graphql_shorts` to your `mix.exs` dependencies:

```elixir
def deps do
  [
    {:graphql_shorts, "~> 0.1.0"}
  ]
end
```

Then, fetch dependencies:

```sh
mix deps.get
```

## Absinthe

This library supports absinthe out-of-the-box, add the dependency:

```elixir
def deps do
  [
    {:absinthe, "~> 1.7"}
  ]
end
```
